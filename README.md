# Calculator
### Arguments:
- `-s` / `--subtract`
- `-a` / `--add`
- `-m` / `--multiply`
- `-d` / `--divide`
### Examples:
`./dist/main --subtract 10 5` -> returns `5`\
`./dist/main --add 10 5` -> returns `15`\
`./dist/main --multiply 10 5` -> returns `50`\
`./dist/main --divide 10 5` -> returns `2` (i think? i failed maths)
